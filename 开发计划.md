## 各 Phase 技术栈

| Phase | 技术栈 | 用途 |
|-------|--------|------|
| **0 - 原型设计** ✅ | HTML + CSS + JS | 纯静态，浏览器直接打开 |
| **1 - 基础设施** ✅ | Python 3.13 / FastAPI / Streamlit / SSE / JSON | 后端骨架 + 前端骨架 + Mock 数据 |
| **2 - Agent 核心** ✅ | LangGraph + LangChain + DeepSeek API + scikit-learn + pandas | Agent 工作流、RFM 聚类、LLM 推理 |
| **3 - Streamlit 前端**  ✅ | Streamlit + plotly | 聊天界面 + 可视化图表 |
| **4 - 联调交付**  ✅ | pytest | 端到端测试 + 交付文档 |

### Phase 2 技术细节

```
LangGraph   ← Agent 工作流编排 (StateGraph + 条件路由 + 中断恢复)
  ├── LangChain    ← 统一 LLM 调用接口 (ChatOpenAI 兼容 DeepSeek)
  ├── ChromaDB     ← 本地向量数据库 (历史文案/活动召回，零配置)
  ├── scikit-learn ← K-Means 聚类 (会员分层)
  └── pandas       ← RFM 指标计算 + 数据结构化
```

> **为什么 Chroma 而不是 Milvus？** MVP 阶段 Chroma 纯 Python 零配置，`pip install` 即用。生产阶段切 Milvus。
